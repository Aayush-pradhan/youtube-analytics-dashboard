# YouTube Analytics Dashboard

A Flask-based web application that analyzes YouTube videos and channels using the YouTube Data API.

## Features

- Video analytics
- Channel analytics
- Engagement rate calculation
- Top channel videos
- Video embed player
- Graph visualization
- Recent search history
- Modern dashboard UI

## Tech Stack

Backend:
- Python
- Flask

Frontend:
- HTML
- Bootstrap

Data Visualization:
- Matplotlib

API:
- YouTube Data API v3

## How It Works

1. User pastes a YouTube video or channel URL
2. The app fetches data using the YouTube API
3. The dashboard analyzes views, likes, and comments
4. Engagement rate is calculated
5. Charts and insights are displayed
